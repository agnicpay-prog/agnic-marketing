# Weekly Growth Dashboard

Owner: the [growth](../skills/growth/SKILL.md) skill. Updated every week as part of the CMO's Weekly GTM Standup (see the [cmo](../skills/cmo/SKILL.md) skill's meeting templates).

Keep this to numbers a founder can read in 30 seconds. Never add a metric that doesn't change a decision.

## Template (fill weekly)

```
WEEK OF: [date]

REVENUE
- New revenue this week: $[...]
- Cumulative MRR/revenue: $[...]

TIER 1 — AI VISIBILITY
- Audits booked: [#]  (cumulative: [#])
- Audits delivered: [#]
- Audit → Optimization conversions: [#]  (conversion rate: [%])
- Active Optimization clients: [#]

TIER 2 — BUILDER PARTNER PROGRAM
- New partner signups: [#]  (cumulative: [#])
- Partners with an active/closed deal: [#]
- Partner-sourced revenue: $[...]

TIER 3 — AGENTIC COMMERCE / THOUGHT LEADERSHIP
- Posts published: [#]
- Notable reach/engagement moment: [one line, only if genuinely notable]

FUNNEL HEALTH
- Top bottleneck this week: [one line]
- One experiment result (win/loss): [one line]

DECISIONS
- What we're stopping: [...]
- What we're doubling down on: [...]
- Next week's #1 priority: [...]
```

## Rules for maintaining this dashboard
- Tier 1 numbers get read first, every week — they're the revenue engine.
- If a number hasn't moved in 2+ weeks, that's the standup's top agenda item, not a footnote.
- Don't add vanity metrics (impressions, followers, page views) unless directly tied to a tier 1/2 conversion step.
