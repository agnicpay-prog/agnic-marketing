# Case Study Template

See the [product-marketing](../skills/product-marketing/SKILL.md) skill for when/how to source these from customer interviews.

## Structure

```
WHO: [Client name/business, one line — real name if permitted, otherwise "a dental clinic in [region]"]

THE PROBLEM (in their words):
[Direct quote or close paraphrase of what was broken for them before Agnic]

WHAT WE DID:
1. [Step — e.g. ran an AI Visibility Audit]
2. [Step — e.g. fixed structured data + directory consistency]
3. [Step — e.g. built a review-generation process]

THE RESULT (one concrete number):
[e.g. "Went from invisible in ChatGPT/Claude/Gemini searches to the #1 recommendation for '[specialty] near [city]' within 3 weeks" / "X new patient inquiries attributed to AI search in the first month"]

THE QUOTE:
"[Direct quote from the client, ideally about the result, in their voice]"
— [Name, Title, Business]
```

## Rules
- No case study ships without a number. If there's no number yet, it's not ready — go get one (revenue, ranking change, inquiries, time-to-result).
- Always get explicit permission before using a client's name/business — use anonymized region-level detail if not.
- Reuse the same case study across cold email, landing page, and LinkedIn — don't write a new one for every channel.
