# LinkedIn Post Template

Voice: founder-led, Dave Gerhardt community-first + Harry Dry hook. See the [community](../skills/community/SKILL.md) skill for strategy, the [copywriter](../skills/copywriter/SKILL.md) skill for line-level rules.

## Structure

```
[Hook line — the most interesting sentence in the post, not a preamble]

[1-3 short paragraphs: the story, lesson, or finding — concrete, specific, real numbers/names where possible]

[Optional: the insight or takeaway, one sentence]

[Optional soft CTA — a question to the audience, or a link, never both]
```

## Post types and when to use them

### 1. Client win / proof post (tier 1)
```
We asked ChatGPT: "best [specialty] near [city]."

[Client]'s practice didn't show up. Not on page 2. Not at all.

Three weeks later, they're the first recommendation.

Here's exactly what we changed: [2-3 concrete fixes]

If you run a [specialty] practice, ask ChatGPT the same question about yourself right now. What do you see?
```

### 2. Founder lesson post (tier 1/2/3 audience)
```
[A specific, slightly uncomfortable truth from the past week — a mistake, a surprising number, a hard call]

[What we did about it]

[What we'd do differently / what we learned]
```

### 3. Builder/partner recruiting post (tier 2)
```
If you build with n8n / LangGraph / MCP and already have clients — here's something you can sell them this week without building anything new.

[one-sentence offer explanation]

[what you earn]

Comment or DM "audit" and I'll send the starter kit.
```

### 4. Thought leadership post (tier 3)
```
[A concrete, non-obvious claim about agentic commerce / AI payments / agent identity]

[Why it matters now, with a specific example]

[What Agnic is building toward this, one sentence, not a pitch]
```

## Checklist before posting
- [ ] Does the first line work if it's the only line someone reads in their feed?
- [ ] Is there a real number, name, or example — not a generality?
- [ ] Would this read the same if written by any other AI startup? If yes, rewrite.
- [ ] One CTA max, and only if it earns its place.
